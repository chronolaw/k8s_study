## Requirements

### Host Machine
| CPU  | Memory | Disk
| -----|--------|-----
| >= 2 | >= 8GB | >= 100GB

### docker/minikube VM
Environment | CPU | Memory | Disk
------------|---- |--------|-----
docker      | >= 2| >= 2GB | >= 20GB
minikube    | >= 2| >= 2GB | >= 20GB

### kubernetes VM
Environment | CPU | Memory | Disk
------------|---- |--------|-----
master node | >= 2| >= 4GB | >= 40GB
worker node | >= 1| >= 1GB | >= 20GB
